# Resources

Place the following image files in this directory:

- `icon.ico` - Windows app icon (256x256 recommended)
- `icon.png` - Linux/Generic app icon (512x512 recommended)
- `icon.icns` - macOS app icon (512x512 recommended)

## Creating Icons

You can use tools like:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [png-to-ico](https://www.npmjs.com/package/png-to-ico)
- Online converters

## Quick Setup

If you don't have icons, the app will use Electron's default icon.

To create a placeholder:
1. Create a 512x512 PNG image
2. Use an online converter to create .ico and .icns files
3. Place them in this directory
